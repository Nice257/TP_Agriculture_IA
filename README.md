#  Prédiction des Récoltes au Burundi

## Description
Application d'Intelligence Artificielle pour prédire les bonnes 
et mauvaises récoltes agricoles au Burundi.

Projet réalisé dans le cadre du TP de BAC 4 Génie Logiciel 
à l'Université Polytechnique de Gitega.

## Technologies utilisées
- Python, Jupyter Notebook
- pandas, numpy, scikit-learn
- matplotlib, seaborn
- Streamlit (application web)

## Comment lancer l'application
```bash
pip install streamlit
streamlit run app.py
```

## Modèles entraînés
- Arbre de Décision
- Forêt Aléatoire  
- Régression Logistique

## Données
- 1620 observations, 15 provinces, 6 cultures (2015–2023)
